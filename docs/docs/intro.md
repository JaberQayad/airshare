# Introduction

AirShare is a **peer-to-peer file transfer application** that lets you share files directly between browsers using WebRTC. We never store your files.

## Features

- **Peer-to-Peer** - Files transfer directly between browsers
- **No Size Limits** - Transfer files of any size
- **Secure** - End-to-end encryption in transit
- **Simple** - Drag and drop, share the link, and you're done
- **Docker Ready** - Optimized for containerized deployments
- **Dark Mode** - Premium UI with native dark mode support

## How It Works

1. Upload files through the web interface
2. Share the generated link with others
3. Files transfer directly peer-to-peer using WebRTC
4. No server storage - your files are private

## Technology Stack

- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Backend**: Node.js, Express
- **Real-time**: Socket.io (Signaling), WebRTC (Data Transfer)
- **Security**: Helmet, CORS, Rate Limiting
